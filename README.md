Un ecosistema visual minimalista basado en la paleta **Nord**, inspirado en el icónico trío **Triple Baka** (Hatsune Miku, Kasane Teto y Akita Neru).

Este ricing está diseñado para ofrecer una experiencia de desarrollo enfocada, silenciosa y estéticamente coherente en entornos Fedora + Sway.

---

## 🛠️ System Components

### 🔵 VS Code
The heart of the environment. Specifically tuned to highlight **Spring Boot** and **Flutter** logic.
- **Base:** Official Nord Theme.
- **Customizations:** Included in the `vscode/settings.json` file.
- **Highlight:** Neru Yellow operators and Teto Red strings/booleans.

### 🔴 Waybar & Wofi
User interface for Sway.
- **Waybar:** Dynamic workspaces (Red for focused, Yellow for urgent/warning).
- **Wofi:** Minimalist launcher with 15px rounded corners and Miku Cyan selection.

### 🟡 Micro & Kitty
Terminal tools synced for a frictionless transition between GUI and CLI.
